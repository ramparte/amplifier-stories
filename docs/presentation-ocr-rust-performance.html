The Solution

Rust + PyO3

  • Python orchestration: File discovery, pattern management, reporting
  • Rust processing: CPU-intensive regex on text
  • PyO3 bridge: Call Rust from Python seamlessly
  • Pre-compiled patterns: Zero overhead on 450+ regexes
  • Rayon parallelization: All 16 cores working

Performance

Where the Speedup Came From

Python Baseline
• Interpreted bytecode
• Regex compiled per file
• Single-threaded (GIL)
• String copying overhead 0.4 files/sec
Rust + PyO3
• Compiled native code (~10×)
• Pre-compiled regex (~5×)
• 16-core parallelism (~2.3×)
• Zero-copy strings (~1.5×) 44.75 files/sec

Multiplier breakdowns are approximate decomposition of the overall 111.9× speedup
