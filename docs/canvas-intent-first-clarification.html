Prompt Engineering
INTENT-FIRST CLARIFICATION
New agent prompt section guiding when and how to ask clarifying questions.
Agent Behavior

Ask WHAT/WHO/WHY, never HOW

The agent is trained to clarify intent, not implementation

Ask About
  • WHAT outcome do you want?
  • WHO is this for?
  • WHY does this matter?
  • WHEN/WHERE context clues
  • Success criteria
Never Ask About
  • "React or Vue?"
  • "Which database?"
  • "What color scheme?"
  • Implementation details
  • Technical preferences

Intent questions unlock the right solution.
Implementation questions just delay it.
