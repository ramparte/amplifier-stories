Lessons Learned

Key Takeaways

1. Native Modules Have Hidden Dependencies
Don't just copy *.node files. Many require companion binaries, data files, or directory structures.
2. Monorepo Build Tools Matter
electron-builder struggles with pnpm workspaces. Use electron-forge for monorepos.
3. macOS PATH Resolution is Special
Apps launched from Finder don't inherit shell PATHs. Always resolve PATH at app startup.
Lessons Learned

More Key Takeaways

4. AI Agents Excel at Systematic Debugging
Methodical approach: trace errors to source ‚Üí inspect file structures ‚Üí test incrementally ‚Üí document findings
5. Automate Early, Not Late
CI/CD took only 12 hours (25% of total effort). Treat release automation as a Day 1 requirement.
6. Draft Releases for Stable, Auto-Publish for Beta
Smart versioning prevents accidental stable releases. Beta tags auto-publish; stable tags create drafts for review.
