## Community Discovery

### Where to Find Community Activity

**GitHub:**
```bash
# Find community repos using Amplifier
gh search repos amplifier --limit 50

# Find discussions
gh api repos/microsoft/amplifier/discussions

# Find contributors
gh api repos/microsoft/amplifier/contributors
```

**Session Data:**
- Monitor public session sharing
- Track bundle downloads
- Identify power users (high session counts)
- Find interesting use patterns

**Social Media:**
- Search for #amplifier, @mentions
- Monitor relevant hashtags
- Track shares and engagement
- Find blog posts about Amplifier

### What to Look For

**Showcase-worthy projects:**
- Novel use cases
- Creative integrations
- Impressive results
- Open source contributions
- Teaching/educational content

**Community contributions:**
- Bug reports with good reproduction steps
- Feature requests with clear use cases
- Documentation improvements
- Helping other users
- Bundle/module contributions

**Engagement opportunities:**
- Questions that need answers
- Discussions worth participating in
- Collaboration opportunities
- Events to promote

## Engagement Metrics to Track

### Community Health Indicators
- New contributors per month
- Repeat contributors
- Discussion participation rate
- Response time to questions
- Positive sentiment in feedback

### Content Engagement
- Blog post views and shares
- Social media engagement (likes, retweets, comments)
- GitHub stars and forks
- Discord activity levels

### Ecosystem Growth
- New public bundles/modules
- Community-created content
- Integrations with other tools
- Use case diversity

## Quality Checklist

Before publishing community content:
- [ ] User/contributor properly credited
- [ ] Links to their work are correct
- [ ] Tone is celebratory and welcoming
- [ ] Permission obtained if needed (for detailed spotlights)
- [ ] Encouragement for others to contribute
- [ ] Call to action included
- [ ] Cross-posted to relevant channels

## Success Criteria

Community content succeeds when:
- Contributors feel appreciated and recognized
- Others are inspired to contribute or share
- Collaboration opportunities emerge
- Community activity increases
- Ecosystem diversity grows
