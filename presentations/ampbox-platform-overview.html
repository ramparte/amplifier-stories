SECURITY

Defense in Depth

Container Security

  • gVisor syscall interception
  • Non-root user (UID 1000)
  • Read-only base filesystem
  • All capabilities dropped

Network Security

  • Network policies enforce isolation
  • Egress-only connections
  • WebSocket origin validation
  • TLS required for API calls

Resource Security

  • CPU/memory quotas per sandbox
  • Ephemeral storage only
  • Auto-cleanup after TTL
  • No persistent data between sessions
QUICK START

Python SDK

Install and start executing code in seconds

Python
# Install the SDK pip install ampbox # Import and create client from ampbox import SandboxClient client = SandboxClient( endpoint="https://sandbox.ampbox.io", api_key="your-api-key", ) # Create sandbox and execute code async with client.create_sandbox(template="python") as sandbox: result = await sandbox.exec("python -c 'print(1+1)'") print(result.stdout) # "2"
QUICK START

TypeScript SDK

Modern async/await API for JavaScript developers

TypeScript
// Install the SDK npm install @ampbox/sdk // Import and create client import { SandboxClient } from '@ampbox/sdk'; const client = new SandboxClient({ endpoint: 'https://sandbox.ampbox.io', apiKey: 'your-api-key', }); // Create sandbox and execute code await client.withSandbox({ template: 'nodejs' }, async (sandbox) => { const result = await sandbox.exec('node -e "console.log(1+1)"'); console.log(result.stdout); // "2" });
